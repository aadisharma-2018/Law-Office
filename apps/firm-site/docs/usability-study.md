# Moderated usability study (SC-003)

## Goal

Show that **≥90%** of representative tasks are completed successfully by **≥5** participants (per **spec** SC-003).

## Script (tasks)

1. Find attorney contact information from the home page.
2. Navigate to **Practice areas** and open one **service** detail page.
3. Locate **Terms** or **Disclaimer** in the footer.
4. Return home using the site header.

Record pass/fail and time-on-task. Note qualitative feedback (confusion, terminology).
